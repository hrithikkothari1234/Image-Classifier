### To clone this project
1) Setup Frontend

```
$ cd frontend_react
$ npm install           
$ npm start              
```

2) Setup Backend

```
$ cd backend_djangoapi
$ pip install requirements.txt
$ python manage.py makemigrations
$ python manage.py migrate
$ python manage.py runserver
```
